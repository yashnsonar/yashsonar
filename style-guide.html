Overall Site Style

The website uses a soft light green background color with the hex value #c0ccc1. The main text color is dark gray with the hex value #333333. The website uses Arial as the main font, with Helvetica and sans-serif as backup fonts.

Text Styling

Most paragraph text is 16px. Paragraphs use a line-height of 1.5 to make the text easier to read. The main page heading uses a font size of 40px and has 2px of letter spacing.

Header

The header is centered and uses dark green text for the main heading. The header has 20px of padding around it to give space between the text and the edge of the page.

Navigation

The navigation area uses a light gray background color with the hex value #e0e0e0. It has a 2px solid black border and 10px of inner padding. Navigation links are blue by default and turn red when the mouse hovers over them. The links do not use underlines because text-decoration is set to none.

Layout and Spacing

The body of the page uses 20px of padding to keep the content away from the browser edges. The navigation uses a bottom margin of 20px to create space below it. The main content area also uses 20px of padding and has spacing below it to separate it from the footer.

Main Content Area

The main section uses a very light gray background color to help separate it from the body background. Section headings in the main area use dark green text. Images use a 2px solid black border and have 15px of space above them.

Footer

The footer is centered and uses the same light gray background color as the navigation. It has 20px of padding and a 2px solid black border at the top to help separate it from the main content.

How to Extend the Style Later

If more pages are added later, they should use the same header, navigation, main, and footer structure so the website stays consistent. New colors, font sizes, or spacing can be changed in the format.css file to update the whole site at once.
