Overall Formatting

The website uses a light green background color (#C0CCC1). The main text color is dark gray (#333). Headings use dark green, and links use blue. The main font-family is Arial, Helvetica, sans-serif. The default paragraph text size is 16px.

Site Header / Branding

The main page heading uses dark green text and is centered on the page. The h1 heading also has 2px letter spacing to make it stand out more. On this style guide page, the heading size is 40px.

Navigation Links

Navigation links use the color blue and do not have underlines because text-decoration is set to none. The navigation section has a black border, a bottom margin of 20px, and list items with 5px vertical spacing. When a user hovers over a link, the link changes to red.

Content Section

The body of the page has 20px padding around all content so the text does not touch the edges of the browser. Paragraphs use a line-height of 1.5 for better readability. Section headings use a top margin of 20px to separate content clearly.

Layout and Spacing

The layout is simple and stacked vertically. The body padding is 20px. The navigation has a margin-bottom of 20px. Each navigation list item has a margin of 5px 0. These spacing choices help keep the page clean and easy to read.

Site Footer

If a footer is added later, it should match the simple site style by using readable text, spacing, and a light background color such as light gray.
